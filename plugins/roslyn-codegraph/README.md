# Roslyn Code Graph Plugin

## Prerequisites

Before installing this plugin, ensure you have:

1. **.NET 10 SDK** — [Download](https://dotnet.microsoft.com/download)
2. **roslyn-codegraph-mcp global tool** — Install with:

```bash
dotnet tool install -g roslyn-codegraph-mcp
```

## Install

```bash
claude install gh:MarcelRoozekrans/roslyn-codegraph-mcp
```
